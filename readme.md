## Generate daily report LaTex tables from schedule and activity files

activities.txt:
Week 1
- Day 1 Activity 1
- Day 1 Activity 2

- Day 2 ...

Week 2
...


schedule.csv:
<DATE>;WORK;<WORKING_HRS> or
<DATE>;HOLIDAY;<HOLIDAY_NOTE> or
<DATE>;SICK or
<DATE>;FLEXITIME


See tex_template.tex how to include the reports in any TeX file
